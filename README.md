# Filtro Sobel en C

Este proyecto aplica el filtro Sobel a imágenes BMP para detección de bordes.

## 📂 Estructura
- input/ → imágenes de entrada
- output/ → resultados generados
- src/ → código fuente

## ▶️ Ejecución

Compilar:
gcc src/main.c -o filtro_sobel -lm

Ejecutar:
./filtro_sobel

## 📥 Uso
Colocar la imagen BMP en la carpeta input/ y escribir su nombre al ejecutar.

Ejemplo:
manga.bmp

## 📤 Salida
Se genera:
output/manga_sobel.bmp
